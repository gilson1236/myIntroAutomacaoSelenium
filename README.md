## Digital Innovation One: Introdução a Automação Web com Selenium e Java

Nesta prática de aulas da DIO, fizemos alguns testes para conhecimento introdutório sobre Automação Web com Selenium baseado nas aulas de Selenium da Digital Innovation One, exibidos
pela Akinely Reis, da Everis. 

Foram desenvolvidos para atingir o objetivo:

* Construção de testes unitários
* Automação destes testes realizados

Utilizei os seguintes requisitos para a execução do projeto: 

* Java 15 
* Maven
* Eclipse, como IDE
* Selenium para automatizar os testes
* Conta no GitHub para armazenar o projeto na nuvem
